# Sample output files

These `.tps` files were generated by this project and verified to load in the
Scalabium Clarion Viewer.

| File | Description |
|------|-------------|
| `arn_sample.tps` | 5 Georgian surnames, 2 fields, 1 key (mirrors a real ORIS table) |
| `accounts_demo.tps` | 6 accounting records: ID, code, name, balance (incl. negatives) |
| `partners_1000.tps` | 1,000 records across multiple pages — stress test |

Open any of them in the Clarion Viewer to confirm record count, structure,
keys, and Georgian text render correctly.
